Voice

Welcome to the guide for voice features in Pycord.

caution

This guide requires previous experience with Pycord and Python.

Requirements​

The only requirement which is required for voice features is PyNaCl. The following features are optional:

tip

Opus is also a required library, but this comes with Pycord. Some hosts may not come with it, though, so remember to always check if required dependencies are installed.

  • FFmpeg - used for sending/receiving files other than .pcm and .wav
  • Pycord.Wavelink, Lavalink.py, Wavelink or any other Python LavaLink library for music playback.
